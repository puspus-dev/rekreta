Telepítés

Egyszerű telepítés

Töltsd le az APK-t, vagy buildeld le magad a forráskódból Flutter segítségével.

1
Engedélyezd az ismeretlen forrásokat Beállítások → Biztonság → Ismeretlen alkalmazások telepítése
2
Töltsd le az APK-t Kattints a letöltés gombra, és nyisd meg a fájlt
3
Telepítsd és jelentkezz be Válaszd ki az iskolád, add meg az OM azonosítód és a jelszavad
# Vagy buildeld le magad
git clone https://github.com/puspus-dev/rekreta
cd rekreta
flutter pub get
flutter build apk --release
